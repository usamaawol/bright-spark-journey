# Bright Spark English Academy

Bright Spark English Academy is a comprehensive English communication academy designed to take students from absolute beginners to confident speakers. We focus on practical, speech-first methods that prioritize real-world communication over rote memorization.

## 🌟 Key Features

### 🎓 Learning Pillars
- **Vocabulary**: Daily words, synonyms, antonyms, and word families with practical examples.
- **Phrases**: Common expressions, idioms, phrasal verbs, and daily communication slang.
- **Pronunciation**: Master IPA, word stress, intonation, and silent letters.
- **Spoken Grammar**: Parts of speech, tenses, and sentence structures built for conversation.

### 🤖 Sparky: Your AI Learning Companion
Enrolled students get exclusive access to Sparky, our AI assistant that provides:
- Instant grammar explanations and writing corrections.
- Real-time pronunciation guidance and speaking practice.
- Personalized study recommendations based on your progress.
- 24/7 assistance with homework and quiz explanations.

### 📊 Student Dashboard
- **My Courses**: Track your journey through progressive levels from Beginner to Champion.
- **Progress Tracking**: Detailed breakdown of your mastery across grammar, vocabulary, speaking, and expressions.
- **Resources**: Access a library of PDF guides, flashcards, audio lessons, and IPA charts.
- **Achievements**: Earn weekly badges and certificates as you advance.

## 🚀 Tech Stack

- **Framework**: [TanStack Start](https://tanstack.com/start) (React + TanStack Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [Radix UI](https://www.radix-ui.com/) + [shadcn/ui](https://ui.shadcn.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Backend/Auth**: [Firebase](https://firebase.google.com/)
- **State Management**: [TanStack Query](https://tanstack.com/query)

## 🛠️ Getting Started

### Prerequisites
- Node.js (v18 or higher)
- Bun (recommended) or npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/usamaawol/bright-spark-journey.git
   cd bright-spark-journey
   ```

2. Install dependencies:
   ```bash
   bun install
   # or
   npm install
   ```

3. Set up environment variables:
   Create a `.env` file in the root directory and add your Firebase and other configuration values.

4. Start the development server:
   ```bash
   bun run dev
   # or
   npm run dev
   ```

5. Build for production:
   ```bash
   bun run build
   # or
   npm run build
   ```
